# Whitsunday Homes — Website

Home builder website for Whitsunday Homes, Airlie Beach QLD.

**Live site:** https://techilaau.github.io/whitsundayhomes

## Structure

```
/                     → Homepage
/home-designs/        → Filterable designs index + individual plan pages
/previous-builds/     → Completed builds gallery
/about/               → About page
/contact/             → Get a Quote
/assets/images/       → Sample images (replace with real photos)
```

## Brand

- Navy `#1C3A4A` · Teal `#4ECDC4` · Deep Teal `#1C8A85` · Sand `#E8DDD0`
- Fonts: Syne (headings) · Josefin Sans (labels) · Outfit (body)
